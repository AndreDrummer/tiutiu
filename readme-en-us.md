## ğŸš€ Technologies Used

**Tiu, Tiu** was built with a set of modern technologies to provide the best user experience:

- **Flutter**: Framework used to ensure a smooth app experience on both Android and iOS.
- **Firebase**: Implemented for authentication, real-time database, push notifications, and more.
  - **Firebase Functions**: For cloud backend logic.
  - **Firebase Dynamic Links**: For dynamic links redirecting to the app.
  - **Firebase Crashlytics**: For error monitoring and fixing.
  - **Firebase Firestore**: Real-time NoSQL database.
  - **Firebase Firestorage**: For image and video storage.
  - **Firebase Hosting**: Used to host the official website.
- **REST API**: For fast and efficient data synchronization.
- **Image and Video Handling**: Media processing within the app, including editing and uploading.
- **Advanced Geolocation**: To find animals near the user.
- **PDF Generation**: To create personalized adoption forms directly within the app.
